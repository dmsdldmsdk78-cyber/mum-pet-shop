# Mum Shop

감성적인 무드와 정돈된 레이아웃을 중심으로 제작한 반려동물 라이프스타일 쇼핑몰 메인 페이지입니다.  
기존의 귀엽고 가벼운 인상보다, 프리미엄 셀렉트숍 느낌에 가깝게 리디자인하는 것을 목표로 작업했습니다.

## Project Overview
- 반려동물 라이프스타일 쇼핑몰 메인 페이지 구현
- 차분한 컬러 톤과 에디토리얼 무드의 UI 구성
- 반응형 대응을 고려한 레이아웃 설계
- 카드형 섹션, 상품 영역, 리뷰 영역, 프로모션 배너 구성

## Features
- 상단 유틸리티 바 및 브랜드 헤더
- 반응형 내비게이션
- 히어로 섹션 이미지 슬라이드
- 카테고리/큐레이션 카드 섹션
- 상품 리스트 섹션
- 리뷰 섹션
- 오프라인 쇼룸 프로모션 배너
- 반응형 UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Responsive
- Desktop
- Tablet
- Mobile

## What I Focused On
- 브랜드형 쇼핑몰 무드에 맞는 여백과 타이포 정리
- 모바일 화면에서 메뉴와 섹션이 무너지지 않도록 반응형 보정
- 단순 상품 나열이 아니라, 라이프스타일 중심의 메인 페이지 흐름 구성
- 기존보다 더 실제 쇼핑몰처럼 보이는 구조로 개선
